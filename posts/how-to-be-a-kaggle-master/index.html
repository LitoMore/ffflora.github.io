Missing values imputation:

  • Numeric Features
    • Impute with mean for normal distributed feature
    • Impute with median for skewed distribution (reduce the impacts of outliers)
  • Categorical Features
    • Majority Imputation
  • Supervised Learning Imputation
    • Regression/Classification
  • XGBoost/LightGBM:
    • Missing values are gracefully treated by considering missing values as candidate for tree splitting point.

Model Ensmble

Stacking is very common in Kaggle community, was a fancy technique, but not fancy any more nowadays.

Ensemble in industry is not common in use, because it is too heavy to maintain, and sometimes it is not very explicable. The only exception in the industry is Facebook is using GBDT+Linear Regression; Linear Regression is fast, but low in accuracy; thus using a combination of GBDT with Linear Regression could make the processing fast at the same time keep the accuracy.
