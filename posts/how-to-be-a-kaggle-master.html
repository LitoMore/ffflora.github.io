Data Preprocessing:

Missing values imputation:

  • Numeric Features
    • Impute with mean for normal distributed feature
    • Impute with median for skewed distribution (reduce the impacts of outliers)
  • Categorical Features
    • Majority Imputation
  • Supervised Learning Imputation
    • Regression/Classification
  • XGBoost/LightGBM:
    • Missing values are gracefully treated by considering missing values as candidate for tree splitting point.

(但不是所有 dataset 和模型都适合 one-hot encoding.)
